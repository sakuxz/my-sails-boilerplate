# my-sails-boilerplate

a [Sails](http://sailsjs.org) application
> 整合 autoreload + babel (async/await) + sequelize + jade + mocha

### Request Environment

npm version >= 3
```bash
# check npm version
npm -v 
# if npm version < 3
npm install npm -g
```

### How to run this app

```bash
npm install
npm start 
# or
sails lift
```

### How to run test

```bash
npm install
npm test
```
